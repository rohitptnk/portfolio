# Rohit Patnaik | Portfolio

A professional, modern, and visually stunning React portfolio website.

## Live Demo
[View Live Portfolio](https://rohitptnk.github.io/portfolio/)

## Tech Stack
- **Framework**: React + Vite + TypeScript
- **Styling**: Vanilla CSS (CSS variables, dynamic glassmorphism, responsive grid)
- **Icons**: Lucide React

## Local Development
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start the local development server

## Deployment
Deployed automatically to GitHub Pages using the `gh-pages` package. 
To deploy a new version:
```bash
npm run deploy
```
